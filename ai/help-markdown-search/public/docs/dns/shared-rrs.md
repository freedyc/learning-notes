共享记录是可以同时应用于多个区中的DNS资源记录，共享记录适用于子域名称、TTL和记录值一致的区记录设置。区中共享记录会有一个小手标记，通过任何位置对共享记录的修改都会在所有共享区生效。

**查询：**  
按照查询文本框灰色字体提示输入要查询的共享记录名称，点击[查询]按钮，系统会将查询结果在共享记录列表中显示，只要名称包含所查询关键词的共享记录都会显示出来，以方便管理员进行下一步操作，默认情况和空查询的结果为全部共享记录。

**共享记录列表：**  
以列表的形式显示当前存在的共享记录信息。信息相关内容包括域名、TTL、记录类型、记录值、共享区。

**新建：**  
填写相应参数完成新建共享记录设置，新建共享记录需要填写以下信息：  
- 共享区：选择新建共享记录所在的区，共享记录成功新建后将在以上所选区中生效。  
- 域名：输入共享记录的子域名称，该域名将分别和各个共享区域名组成全称域名。  
- 类型：通过下拉列表框选择记录类型，共享记录支持的类型均在下拉列表中提供。  
- TTL：填写该共享记录的TTL，TTL为允许其它服务器将记录在缓存中存放的时间。  
- 记录值：填写该共享记录类型对应的值信息。  

**编辑：**  
当单个勾选上列表中记录前面的复选框时，此功能可用，没有任何勾选或者勾选多于1个时，此功能不可用。编辑单个记录不能修改记录的域名、类型，可以修改TTL和记录值，设置规则同新建。{{.ADD_LICENSE:每次编辑单条共享记录的非宕机切换参数，须逐个手动重新添加其它区该共享记录的宕机切换策略。}}

**修改所属区：**  
当单个勾选上列表中记录前面的复选框时，此功能可用，没有任何勾选或者勾选多于1个时，此功能不可用。在所属区选项中删除或者选择区，以修改该共享记录的所属区。

**删除：**  
当勾选中列表中区前面的复选框时（可多选），此功能可用，没有任何勾选，此功能不可用。

（注：通过右键点击共享记录列表中不同位置，会有相应菜单显示）
